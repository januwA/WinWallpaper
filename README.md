# WinWallpaper

在windows将video设置为壁纸

## 使用
1. 在文件中*选择*视频
2. 点击*播放*
3. 播放后点击*置底*按钮

- 视频播放使用ffplay
- 有些时候可能占用极高CPU(可能和视频有关,视频帧率过高如60fps，通常作为壁纸30fps足够了)
- [转换视频工具](https://github.com/januwA/WPF-ffmpegtool)
